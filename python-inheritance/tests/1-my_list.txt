The MyList Class
================

Import the module:
    >>> MyList = __import__('1-my_list').MyList

Test 1: Simple list of integers
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Test 2: Negative numbers
    >>> my_list2 = MyList()
    >>> my_list2.append(-1)
    >>> my_list2.append(-5)
    >>> my_list2.append(0)
    >>> my_list2.print_sorted()
    [-5, -1, 0]

Test 3: Empty list
    >>> my_list3 = MyList()
    >>> my_list3.print_sorted()
    []
